# How to Run the Quiz Application

## Quick Start

### Method 1: Using Python directly
Open your terminal/command prompt in the project directory and run:

```bash
python src/main.py
```

### Method 2: Using PyCharm
1. Open the project in PyCharm
2. Right-click on `src/main.py`
3. Select "Run 'main'"

## What You'll See

When you run the application, you'll see the main menu with these options:

```
============================================================

============================================================
```

## How to Navigate

1. **Type a number** (0-10) to select a menu option
2. **Press Enter** to confirm your choice
3. **Follow prompts** to complete actions
4. **Type 0** to exit the application

## Main Features

### 1. Create Question
- Create multiple choice, true/false, or select-all questions
- Add answer options
- Assign tags to organize questions

### 2. Take Quiz
- Start a new quiz
- Answer questions
- See immediate feedback
- View your score at the end

### 3. Manage Tags
- Create and organize tags
- View tag hierarchy
- Search and filter by tags

### 4. Enhanced Question Management
- Browse all questions
- Edit existing questions
- Bulk operations
- Version control

### 5. Question Types
- View question type statistics
- Manage question templates
- Convert between question types

### 6. Database Management
- View database statistics
- Create backups
- Restore from backups
- Database maintenance

### 7. Analytics Dashboard
- View performance statistics
- See quiz history
- Track learning progress

### 8. Import from Screenshot
- Import questions using OCR
- Process images to extract questions

### 9. Settings
- Configure application preferences
- Adjust display options

### 10. Help
- View help documentation
- See keyboard shortcuts
- Get usage tips

## Example Usage Flow

1. **Create a Question**:
   - Select option `1`
   - Choose question type (multiple choice, true/false, or select-all)
   - Enter question text
   - Add answer options
   - Mark correct answer(s)
   - Add tags

2. **Take a Quiz**:
   - Select option `2`
   - Choose quiz type (random, by tags, or custom)
   - Answer questions as they appear
   - View feedback immediately
   - See final score when complete

3. **Exit**:
   - Select option `0` from any menu
   - Or press `Ctrl+C` to quit

## Tips

- All menus are numbered for easy navigation
- You can go back to previous menus by selecting the "Back" option
- Invalid inputs will show error messages and prompt again
- The application saves your data automatically

Enjoy using the Quiz Application!
